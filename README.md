# 📋 To-Do List

A simple and handy To-Do List app for managing your tasks.

Created for practicing **HTML**, **CSS**, and **JavaScript** skills.

---

## 📸 Screenshot

![Screenshot](screenshot.jpg)

---

## 🚀 Features

✅ Add new tasks  
✅ Set due dates  
✅ Mark tasks as completed  
✅ Delete tasks  
✅ Save tasks in localStorage  
✅ Smooth animations  
✅ Dark theme
